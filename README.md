## Run Locally

Clone the project

```bash
  git clone https://github.com/movielust-in/movielust-next.git
```

Go to the project directory

```bash
  cd movielust-next
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
